# cli-tools

A collection of small CLI utilities for working with Kubernetes and AWS.

## Tools

| Tool | Description |
|------|-------------|
| [kubeenv](kubeenv/) | Interactive TUI for switching `kubectl` contexts |
| [aws-op](aws-op/) | Wrapper that pulls AWS credentials from 1Password for EKS token retrieval |
